## Logo

The logo is a simple version of the boklisten logo

## Searchbar

The searchbar is where you can search for customers. Either by `name`, `email`,
`phone` or `address`.

The results will be shown in a dropdown-list navigational by arrow-keys.

## Branch select

All the way to the right you can select a branch.

## Location in code

The header code is located in `src/app/header/`
