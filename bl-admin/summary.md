# Introduction

`bl-admin` is a comprehensive admininstation tool for boklisten.no's handling
of orders, customers, invoices and much more. It is at it's core a
order-administration application, handeling delivery, handout and payment.

# Functionality

There are loads of functionality in `bl-admin`. Under is a list with some of them:
* Order handling
* Invoice creation
* Sending of reminders
* Customer detail edit
* Customer search
* Uploading of Items

# Goal

To be easy to use, and self-explanatory. It should be no need for a deep-dive
into a long documentation for a simple function.
